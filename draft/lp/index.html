✦ Features

すべてが ブラウザで完結

コマンドラインは不要。直感的なWeb UIでネットワーク構築からモニタリングまで。

⚡

ワンクリック起動

docker compose up ひとつでバックエンド・フロントエンド・Bootstrap が自動セットアップ。複雑な構成は不要です。

🎛️

ビジュアル設定エディタ

100以上のネットワークパラメータをカテゴリ分けされたフォームで編集。ブロック生成時間、モザイク、ネームスペースなど自在に設定。

📊

リアルタイムダッシュボード

ノードヘルス、チェーン高、ピア数、ファイナライズ状態を WebSocket でリアルタイム表示。ターミナルログもライブで確認。

🔍

内蔵エクスプローラー

ブロック・トランザクション・アカウントを視覚的に閲覧。bootarou/explorer-smd を Docker で自動ビルド＆起動。

🤝

ネットワーク共有

構築したネットワーク設定をZIPパッケージでエクスポート。他のメンバーはインポートするだけでノード参加可能。

🔐

アドレス管理

暗号化された秘密鍵の復号表示、全アドレスの残高確認、アドレスファイルのダウンロードに対応。

✦ How it Works

3ステップで起動

ZIP を展開

ダウンロードしたZIPを展開し、.env ファイルを準備するだけ。

Docker で起動

docker compose up -d を実行。ビルドからセットアップまで全自動。

ブラウザでアクセス

http://localhost:5173 を開いて、ネットワーク設定・起動・管理を開始。
